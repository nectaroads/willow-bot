# Sobre
- Willow.js é um BOT de discord capaz de moderar, gerenciar e divertir servidores da plataforma; Manipular bancos de dados; Controlar servidores dedicados de jogos (focada em Don't Starve Together); Executar comandos; Dentre outros.
- A proposta do BOT é criar uma assistente capaz de deixar a manipuação de servidores dedicados dinâmica (Um dos projetos pessoais é um futuro MMORPG)

# Conteúdo
- Moderação: Possui o poder de moderar servidores do Discord, executando medidas como: banimentos; expulsões; dentre outros.
- Diversão: Willow é capaz de divertir membros através de memes, comandos bobos e joguinhos.
- Servidores Dedicados: O bot é capaz de iniciar servidores dedicados de Don't Starve Together, além de manipulá-los.
- Banco de Dados: Para adicionar mais imersão em seus servidores, possui em seu kit um banco de dados que alimenta um rank, pontuações, prêmios e etc.
